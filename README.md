# snow

### 介绍

snow 是 go 语言服务器应用的通用解决方案，游戏服务器是 snow 的典型应用场景

> 项目经过验证，但很多设计仍未完成，需要大幅调整，因此暂不考虑版本兼容性（即版本为 0.x.y）

### 模块简介

- `container` 常用泛型容器及容器通用算法
- `core` 其他核心基础库
- `injection` 依赖注入的接口
- `host` 依赖注入的 host 框架实现，用于标准化应用启停流程，类似 .net 的 hosting
- `node` 基于 host 框架的 actor 框架
